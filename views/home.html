Pandemic Site (under construction)


This is a web application that I co-created which shows the real-time spread of disease. Our goal was to make a complex algorithm that takes into account speed, location proximity, disease resistance and strength. One feature that I focused on was spatial hashing which drastically reduced the computations per frame by over 95%.

Skills used: Angular 1x, HTML5, CSS3, JavaScript, Canvas, Firebase, Github, and Gulp.

www.projectpandemic.com
